Mecánicas de UsuarioUser Mechanics

🔥 Racha diariaDaily streak

Días consecutivos practicando. Récord personal guardado en perfil.

Consecutive days of practice. Personal best saved in profile.

😴 Prompt de distracciónDistraction prompt

Tiempo expirado sin respuesta → ¿Estabas distraído? A = no cuenta / N = cuenta como error.

Time up with no answer → Were you distracted? A = skip / N = count as wrong.

⏳ Penalización por abandonoNeglect penalty

Sin practicar ≥5 días con nivel >1 → baja un nivel al abrir la app.

No practice for ≥5 days at level >1 → drops one level on app open.

⏱️ Tiempo adaptativoAdaptive time limit

acc >90% → 50% del tiempo base. acc <40% → 140%. Más dominio, más presión.

acc >90% → 50% of base time. acc <40% → 140%. More mastery, more pressure.
